# HTML

## 如何理解 HTML 语义化

    • 让人更容易读懂（增加代码可读性）
    • 让搜索引擎更容易读懂（更好的 SEO）

举例：

```html
<div>
  <h1>标题</h1>
  <p>文字</p>
  <ul>
    <li>第一项</li>
    <li>第二项</li>
    <li>第三项</li>
  </ul>
</div>
```

## 块状元素&内联元素？（html、css 结合，常考）

    • display: block/table;
              有 div h1 h2 table ul ol p 等（独占一行）
    • display: inline/inline-block;
              有 span img input button 等（不会独占一行，挤到换行）
